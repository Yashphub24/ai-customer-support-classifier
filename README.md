# AI Customer Support Classifier

An NLP-based customer support ticket classification system developed using Python. The project automatically categorizes customer support requests into five predefined categories: Refund Request, Technical Issue, Cancellation Request, Product Inquiry, and Billing Inquiry.

The system processes customer ticket text, applies text classification techniques, and routes tickets to the appropriate support category. This helps reduce manual effort, improve response times, and streamline customer support operations.

### Key Features

* Automatic support ticket categorization
* Classification of 8,469 customer support tickets
* NLP-based text processing
* Keyword-based ticket routing
* Performance evaluation using accuracy metrics
* Scalable solution for customer support automation

### Technologies Used

* Python
* Natural Language Processing (NLP)
* Pandas
* NumPy
* Scikit-learn

### Dataset Information

* Total Tickets: 8,469
* Categories: 5

  * Refund Request
  * Technical Issue
  * Cancellation Request
  * Product Inquiry
  * Billing Inquiry

### Outcome

The classifier successfully automated ticket categorization, demonstrating how NLP can be used to improve support ticket routing efficiency and reduce manual workload.
